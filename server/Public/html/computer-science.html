CSE - 400

Fundamentals of Probability in Computing

This course explores the use of probability theory in computer science, including randomized algorithms, probabilistic analysis, and statistical modeling. It covers concepts like probability distributions, expectations, and variance, with applications in data analysis, machine learning, and performance evaluation of algorithms.

CSE - 302

Computer Organization and Architecture

This course examines the internal structure and functioning of computer systems, focusing on how hardware and software interact. Topics include CPU architecture, memory hierarchy, input or output mechanisms, instruction sets, and performance optimization techniques, providing a foundational understanding of how computers execute programs.

CSE - 301

Design and Analysis of Algorithms

This course delves into the design and analysis of efficient algorithms to solve computational problems. It covers fundamental techniques like divide-and-conquer, dynamic programming, greedy algorithms, and graph algorithms. Emphasis is placed on time and space complexity, helping students develop optimized solutions for complex problems.

CSE - 100

Fundamentals of Computer Programming

This course introduces the basics of programming and problem-solving techniques. It covers essential programming concepts such as variables, data types, control structures (loops and conditionals), functions, and arrays. Students learn to write, debug, and optimize code while developing logical thinking and computational skills using a beginner-friendly programming language.

CSE - 203

Object oriented Programming

This course covers the fundamentals of object-oriented programming, including concepts like classes, objects, inheritance, polymorphism, encapsulation, and abstraction. It focuses on designing modular and reusable code using real-world scenarios and problem-solving techniques.

CSE - 205

Data Structures and Algorithms

This course explores essential data structures such as arrays, linked lists, stacks, queues, trees, and graphs. It also covers algorithm design and analysis, including sorting, searching, and recursion, with a focus on optimizing time and space complexity.

CSE - 213

Digital Design with HDL

This course delves into the principles of digital logic and circuit design. It includes topics such as Boolean algebra, combinational and sequential circuits, flip-flops, counters, and memory units. It also emphasizes the use of HDL (Hardware Description Language) for implementing digital systems.

CSE - 250

Database Management Systems

DBMS focuses on the design, implementation, and management of databases. It covers concepts like data modeling, normalization, SQL queries, transaction management, and indexing. It ensures efficient data storage, retrieval, and security, supporting multi-user environments.

CSE - 342

Computer networks

This course explores the structure and functioning of computer networks, including the OSI and TCP/IP models. It covers protocols, routing, switching, network security, and wireless communication. It also includes hands-on experience with network configurations and simulations.

CSE - 212

Cybersecurity

Cybersecurity involves protecting computer systems, networks, and data from unauthorized access, attacks, and damage. The course covers encryption, network security protocols, ethical hacking, malware analysis, and risk management. It emphasizes building secure systems and responding to cyber threats.

CSE - 199

Music Throught Computing

This interdisciplinary course combines computing with music, exploring digital sound synthesis, music programming languages, and algorithmic composition. It covers audio processing, music visualization, and the development of interactive musical systems.

CSE - 211

Discrete Mathematics

Discrete Mathematics includes topics essential for computer science, such as logic, set theory, combinatorics, graph theory, and number theory. It forms the foundation for algorithms, data structures, and programming language design.

CSE - 602

Deep Learning

Deep Learning focuses on neural networks and advanced architectures like CNNs, RNNs, and transformers. It covers backpropagation, optimization techniques, and applications in image recognition, natural language processing, and generative models.

CSE - 623

Machine Learning Theory and Practice

This course covers fundamental machine learning algorithms, including supervised, unsupervised, and reinforcement learning. It focuses on model evaluation, bias-variance trade-off, feature engineering, and hands-on implementation using tools like Scikit-learn and TensorFlow.

CSE - 641

Computer Vision: Modern Methods and Applications

Computer Vision explores techniques for image processing, object detection, and scene understanding. It covers modern methods like convolutional neural networks (CNNs), transfer learning, and applications in autonomous vehicles, facial recognition, and augmented reality.

CSE - 533

Social Network Analysis

This course studies the structure and dynamics of social networks using graph theory and statistical models. It covers community detection, influence propagation, network visualization, and applications in sociology, marketing, and information diffusion.

CSE - 519

Human Computer Interaction

HCI examines the design and evaluation of user interfaces to enhance user experience. It covers usability principles, interaction design, user-centered design processes, prototyping, and testing, with a focus on accessibility and aesthetics.

CSE - 542

Introduction to Blockchain: Technologies, Approaches and Applications

This course introduces blockchain technology, covering its underlying principles like decentralized ledgers, cryptographic security, and consensus mechanisms. It explores smart contracts, cryptocurrencies, and applications beyond finance, including supply chain and healthcare.
